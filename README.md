## 🎮 게임 소개

**Vegetable Guardians**는 끊임없이 밀려오는 적들을 물리치며 강력한 능력을 수집하는 액션 슈팅 게임입니다. 
웨이브마다 새로운 적들이 등장하고, 레벨업 시 다양한 능력을 선택하여 나만의 최강 전투 스타일을 만들어보세요!

### ✨ 주요 특징

#### 🎯 **100% 조합 가능한 능력 시스템**
- 모든 능력이 서로 조합되어 시너지 발생
- 13가지 멀티샷 조합 가능 (펜타샷 + 쿼드샷 + 트리플샷 + 더블샷 = 13발!)
- 특수 탄환 속성 중첩 가능 (유도 + 폭발 + 관통 + 연쇄번개 동시 적용)

#### 🌊 **웨이브 기반 진행 시스템**
- 각 웨이브마다 고정된 수의 적 처치 필요
- 웨이브가 진행될수록 새로운 적 등장
- **5웨이브마다 강력한 보스 출현!**

#### 👾 **다양한 적과 보스**
- 6가지 적 타입, 각자 고유한 움직임과 공격 패턴
- 7가지 이동 패턴 (직선, 사인파, 지그재그, 원형, 대각선, 돌진, 웨이브)
- 6가지 보스 몬스터와 특별 보상

#### 📱 **모바일 & PC 완벽 지원**
- 터치 조작 완벽 지원
- 자동 발사 시스템
- 반응형 디자인

## 🎯 게임 플레이 가이드

### 🚀 **게임 시작하기**
1. **캐릭터 선택**: 5가지 야채 캐릭터 중 선택 (🥕🌽🥬🥦🥒)
2. **적 처치**: 자동으로 발사되는 탄환으로 적 공격
3. **경험치 획득**: 적을 처치하면 경험치 획득
4. **레벨업**: 경험치가 가득 차면 능력 선택 화면 등장

### 💪 **능력 선택 전략**

#### 🏆 **능력 등급 시스템**
능력은 7개 등급으로 나뉘며, 각 등급마다 고유한 확률로 등장합니다:

| 등급 | 설명 | 출현 확률 |
|------|------|----------|
| **SSS** | 🌟 전설급 능력 | 2% |
| **SS** | ⭐ 영웅급 능력 | 5% |
| **S** | ✨ 희귀 능력 | 10% |
| **A** | 💫 고급 능력 | 18% |
| **B** | ⚡ 일반 능력 | 25% |
| **C** | 🔹 기본 능력 | 25% |
| **D** | 🔸 시작 능력 | 15% |

#### 📊 **추천 빌드**

##### 🔥 **폭발 빌드** (광역 화력)
```
1단계: 더블샷 (D)
2단계: 폭발 탄환 (B)
3단계: 펜타샷 (A)
4단계: 메가 폭발 (S) - 폭발 범위 +100%
5단계: 멀티샷 (A)
보스: Inferno Blast (폭발 범위 +100%)
결과: 6발 × 여러번 × 200px 폭발 = 화면 전체 폭발
```

##### ⚡ **연쇄 빌드** (다수 타겟)
```
1단계: 관통 (C)
2단계: 연쇄 번개 (A)
3단계: 멀티샷 (A)
4단계: 체인 업그레이드 (S)
5단계: 쿼드샷 (A)
보스: Chain Master (연쇄 +2번, 범위 +50%)
결과: 여러발 × 3명 관통 × 5번 튕김 = 적 전멸
```

##### 🎯 **정밀 빌드** (100% 명중)
```
1단계: 유도 미사일 (B)
2단계: 치명타 (C)
3단계: 나선 탄환 (B)
4단계: 폭발 탄환 (B)
5단계: 펜타샷 (A)
보스: Perfect Aim (유도 + 치명타)
결과: 100% 명중 × 40% 크리티컬 × 폭발 = 완벽한 화력
```

##### 🌀 **카오스 빌드** (모든 능력 조합)
```
목표: 모든 멀티샷 + 모든 특수 속성
펜타 + 쿼드 + 트리플 + 더블 = 13발
+ 유도 + 나선 + 관통 + 폭발 + 치명타 + 연쇄번개 + 냉동 + 부메랑
결과: 완전무적 상태
```

## 🎯 게임 팁

### 💡 **초보자를 위한 팁**
1. **초반 생존이 우선**: 체력 관리를 잘하세요
2. **멀티샷 먼저**: 더블샷 → 트리플샷 순서로 획득
3. **속성 조합**: 유도 + 폭발 조합이 강력합니다
4. **보스 준비**: 5웨이브 전에 능력 강화 필수

### 🏆 **고수를 위한 팁**
1. **시너지 극대화**: 모든 멀티샷 + 모든 특수 속성 조합
2. **조건부 능력**: 폭발 보유 시 메가 폭발 노리기
3. **보스 보상 전략**: 부족한 능력 위주로 선택
4. **적 패턴 학습**: 각 적의 움직임 예측하기

### ⚠️ **주의사항**
- 적이 화면 하단에 도달하면 **맨 위에서 다시 스폰**됩니다 (데미지 없음)
- 레벨업 선택지에서 **같은 능력은 중복 등장하지 않습니다**
- 능력은 **모두 중첩 가능**하며, 순서에 관계없이 적용됩니다

## 🛠️ 기술 스택

### 🎨 **프론트엔드**
- **Vanilla JavaScript (ES6+)**: 순수 자바스크립트로 구현
- **HTML5 Canvas API**: 고성능 2D 렌더링
- **CSS3**: 반응형 디자인 및 애니메이션
- **ES6 Modules**: 모듈화된 코드 구조

### 🏗️ **아키텍처**
```
Vegetable-Guardians/
├── index.html              # 메인 HTML
├── styles.css              # 전역 스타일 (모바일 최적화)
└── js/
    ├── main.js            # 게임 진입점
    ├── Game.js            # 게임 로직 컨트롤러
    ├── Player.js          # 플레이어 클래스
    ├── Enemy.js           # 적 클래스
    ├── Boss.js            # 보스 클래스
    ├── Bullet.js          # 탄환 클래스
    ├── Particle.js        # 파티클 효과
    ├── GameObject.js      # 기본 게임 객체
    ├── config.js          # 게임 설정 상수
    └── utils.js           # 유틸리티 함수
```

### 🎯 **핵심 기술**

#### 1️⃣ **객체 지향 프로그래밍 (OOP)**
- `GameObject` 기반 상속 구조
- 각 게임 요소를 클래스로 모듈화
- 재사용 가능한 컴포넌트 설계

#### 2️⃣ **게임 루프 (Game Loop)**
```javascript
requestAnimationFrame() // 60 FPS 최적화
```
- 부드러운 60 FPS 렌더링
- deltaTime 기반 프레임 독립적 움직임
- 효율적인 업데이트/렌더링 분리

#### 3️⃣ **충돌 감지 (Collision Detection)**
```javascript
// AABB (Axis-Aligned Bounding Box) 알고리즘
checkCollision(obj1, obj2)
```
- 사각형 기반 충돌 감지
- 효율적인 다중 객체 충돌 처리

#### 4️⃣ **이벤트 시스템**
- 키보드 입력 (keydown/keyup)
- 터치 입력 (touchstart/touchmove/touchend)
- 캔버스 좌표 정규화

#### 5️⃣ **상태 관리**
```javascript
states: menu, playing, paused, gameover
```
- FSM (Finite State Machine) 패턴
- 게임 상태별 렌더링 최적화

#### 6️⃣ **능력 시스템**
- 동적 능력 조합 알고리즘
- 가중치 기반 확률 시스템
- 조건부 능력 필터링

#### 7️⃣ **파티클 시스템**
```javascript
ExplosionEffect, TextEffect, DamageNumber
```
- Canvas를 활용한 시각 효과
- 효율적인 파티클 수명 관리
